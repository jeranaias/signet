# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

Be decent. Disagreement is welcome; harassment is not. The maintainers will remove or block contributors whose behavior makes other people's work harder.

## Reporting

Conduct concerns: jesse@thornveil.ai. Reports are handled privately.

## Scope

Applies to all project spaces: GitHub issues, PRs, discussions, the project's PyPI page, and any communication done in a signet maintainer capacity.
